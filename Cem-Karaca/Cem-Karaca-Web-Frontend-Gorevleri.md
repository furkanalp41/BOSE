Piyasa Emri Oluşturma Sayfası
API Endpoint: POST /orders/market Görev: Anlık piyasa fiyatından alım-satım panelinin tasarımı UI Bileşenleri: Al (Yeşil) ve Sat (Kırmızı) sekmeleri Miktar (Lot/Adet) input alanı İşlem tutarı özeti (Miktar * Anlık Fiyat) Büyük ve dikkat çekici "Alış Emri Gönder" butonu Form Validasyonu: Miktar 0'dan büyük olmalı Kullanıcının bakiyesi işlem tutarını karşılamıyorsa buton disabled olmalı Kullanıcı Deneyimi: Geçersiz miktar girildiğinde input altında kırmızı uyarı Başarılı işlemde konfeti efekti veya success toast mesajı Teknik Detaylar: Gerçek zamanlı fiyat verisiyle işlem tutarının anlık hesaplanması Number formatting (Binlik ayraçlar ve ondalık kısımlar)
Limit Emir Oluşturma Formu
API Endpoint: POST /orders/limit Görev: Özel fiyatlı ileri tarihli emir giriş formu UI Bileşenleri: Hedef Fiyat input alanı (artır/azalt okları ile) Miktar input alanı Toplam Tutar göstergesi Kullanıcı Deneyimi: Anlık fiyatın %10 altı/üstü gibi hızlı seçim butonları Inputlara sadece rakam ve nokta girilebilmesi Teknik Detaylar: Controlled input yönetimi ve decimal hassasiyet (precision) ayarları
Açık Emirleri Listeleme Paneli
API Endpoint: GET /orders/open Görev: Bekleyen emirlerin (henüz gerçekleşmemiş) listelendiği tablo UI Bileşenleri: Emir Tipi, Hedef Fiyat, Miktar, Tarih sütunları Her satırın sonunda "İptal Et" butonu Liste boşsa "Bekleyen emriniz yok" yazan empty state Kullanıcı Deneyimi: Tablo satırlarının üzerine gelince (hover) belirginleşmesi Teknik Detaylar: Date formatting (Tarih ve saatin yerel formata çevrilmesi) Düzenli aralıklarla verinin arka planda güncellenmesi
Limit Emir Güncelleme Modal'ı
API Endpoint: PUT /orders/{orderId} Görev: Bekleyen emrin fiyat veya miktarını düzenleme arayüzü UI Bileşenleri: Satırdaki "Düzenle" butonuna basınca açılan Modal Eski verilerle dolu fiyat ve miktar inputları "Değişiklikleri Kaydet" butonu Kullanıcı Deneyimi: Güncelleme sonrası Modal'ın otomatik kapanması ve tablonun yenilenmesi Teknik Detaylar: Modal portal kullanımı ve form state'inin emir ID'sine göre bağlanması
Bekleyen Emri İptal Etme İşlemi
API Endpoint: DELETE /orders/{orderId} Görev: Emri iptal edip bakiyeyi boşa çıkarma akışı UI Bileşenleri: Satır sonundaki çarpı (X) veya çöp kutusu ikonu Kullanıcı Deneyimi: İptal butonuna basıldığında satırın kaybolma animasyonu (fade out) İade edilen bakiyenin üst menüde anında güncellenmesi Teknik Detaylar: DELETE isteği başarılıysa lokal state'ten ilgili ID'nin filtrelenmesi
AI Portföy Raporu Dashboard'u
API Endpoint: GET /ai/report/portfolio/{userId} Görev: Yapay zekanın kullanıcı portföyünü analiz ettiği dashboard tasarımı UI Bileşenleri: Portföy dağılımını gösteren interaktif Pasta Grafiği (Pie Chart) Yapay zeka risk skoru (1-100 arası bar grafik) AI strateji önerisi metin kutusu Kullanıcı Deneyimi: Sayfa yüklendiğinde grafiklerin sıfırdan dolma animasyonu Grafik dilimlerinin üzerine gelince (tooltip) hisse detaylarının çıkması Teknik Detaylar: Recharts, Chart.js veya Highcharts kütüphanesi entegrasyonu Karmaşık JSON verisinin grafik formatına dönüştürülmesi
